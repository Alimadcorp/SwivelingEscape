\# Swiveling Escape



This game was made in Blender Game Engine





Blender Game Engine was discontinued after v2.79b, therefore, download it here:



https://download.blender.org/release/Blender2.79



\## To edit:



Open `project.blend` in Blender 2.79b or older



If it isn't already, switch the engine to `Blender Game`



Press `P` to play, `Esc` to stop playinh





\## To build:



Enable the `Game Engine Runtime` addon in the addon preferences



Go to `File > Export > Game Engine Runtime` and select a path to export to



The folder will get saturated with a bunch of dlls, and a game executable
